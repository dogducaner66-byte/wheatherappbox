# Weatherbox

Weatherbox is a Vite + React + TypeScript weather app starter focused on a premium mobile-first shell, persisted user controls, location search, saved cities, and browser geolocation fallbacks.

## Scripts

- `npm test` runs the Vitest suite.
- `npm run lint` runs ESLint.
- `npm run build` creates the production bundle.

## Stack

- Vite + React + TypeScript
- Tailwind CSS v3
- Framer Motion
- Zustand
- TanStack Query v5
- Vitest + React Testing Library
